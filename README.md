# Learnify

A learning platform built with Spring Boot.

## Tech Stack
- Java 21
- Spring Boot 3.5.11
- Maven
- H2 Database
- Spring Data JPA
- Thymeleaf

## Setup
1. Clone the repo
   git clone https://github.com/Plooozy/learnify
2. Open in IntelliJ IDEA
3. Run LearningPlatformApplication.java
